Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • HTML elements or Class attributes can be used as selectors when styling.
  • Use the Link element to add the Style Sheet to the Head section in the HTML file.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file, and the rules only apply to that page.
  • External style sheets are the most commonly used and need to be linked to the HTML file. Keeps code separate.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds new changes made in working branch.
  • git commit -m "change notes": commiits changes in working branch.
  • git push origin main: pushes commited changes to main branch.
